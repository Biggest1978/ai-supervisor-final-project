# Краткое описание проекта

## Название проекта

`AI Supervisor`

## Что это

`AI Supervisor` — это проектный AI-агент для анализа диагностическо-продажных консультаций.

Он нужен для того, чтобы разбирать консультации, видеть повторяющиеся ошибки экспертов, фиксировать сильные и слабые паттерны, формировать рекомендации по следующему шагу и собирать аналитический слой для будущего CRM-контура.

## Какую проблему решает

Когда контроль качества консультаций делается вручную, он плохо масштабируется:

- нельзя стабильно отсматривать много встреч;
- сложно увидеть системные ошибки;
- тяжело превращать наблюдения в единый обучающий контур;
- часть полезной аналитики теряется и не возвращается в CRM.

`AI Supervisor` решает эту проблему за счет проектной агентной логики и структурированных выходных артефактов.

## Что делает MVP

MVP позволяет:

- анализировать консультации;
- формировать review по встречам;
- собирать аналитические выводы;
- поддерживать отчетный слой;
- накапливать знания для будущей `CRM Dream`.

## Почему это итоговый проект, а не учебный пример

Проект привязан к реальной рабочей задаче и является частью более широкого контура `CRM Dream` / цифровой экосистемы отдела диагностик.
