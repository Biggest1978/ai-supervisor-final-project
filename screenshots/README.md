# Скриншоты для сдачи

Сюда можно добавить 3-5 скриншотов для итоговой упаковки проекта.

## Рекомендуемый набор

1. Главная страница репозитория.
2. Открытый `README.md`.
3. Открытый `AGENT_ARCHITECTURE.md`.
4. Открытый `SUBMISSION_MESSAGE_RU.md`.
5. При желании — скриншот списка файлов репозитория как доказательство упаковки проекта.

## Зачем это нужно

Скриншоты помогают закрыть формальную часть домашнего задания:

- показать, что проект опубликован в GitHub;
- показать, что у проекта есть описание;
- показать, что есть логика проекта и материалы для защиты.
