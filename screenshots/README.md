# Скриншоты для сдачи

В этой папке лежат реальные скриншоты публичной упаковки проекта.

## Текущий набор

1. `01_repo_home.png` — главная страница репозитория.
2. `02_readme_overview.png` — основной экран `README`.
3. `03_readme_file_view.png` — просмотр `README.md` как отдельного файла внутри репозитория.

## Зачем это нужно

Скриншоты помогают закрыть формальную часть домашнего задания:

- показать, что проект опубликован в `GitHub`;
- показать, что у проекта есть описание;
- показать, что репозиторий оформлен и готов к защите.

## Что можно добавить при желании

- скриншот `AGENT_ARCHITECTURE.md`;
- скриншот `SUBMISSION_MESSAGE_RU.md`;
- скриншот списка файлов репозитория целиком.
