# Короткий скрипт защиты

Я сделал итоговый проект `AI Supervisor`.

Это агентная система, которая помогает анализировать диагностическо-продажные консультации, выявлять ошибки экспертов, собирать отчетность и возвращать аналитические выводы в будущий CRM-контур.

Я выбрал эту тему, потому что она решает реальную рабочую задачу. Вручную разбирать консультации долго и тяжело масштабировать. Мне было важно сделать не учебный пример, а прикладной MVP.

Внутри проекта есть:

- проектная агентная логика;
- понятная структура артефактов;
- слой анализа консультаций;
- слой отчетности;
- связь с `CRM Dream`.

Если очень коротко показать логику проекта, она выглядит так:

`транскрипт -> review -> аналитический вывод -> рекомендации -> данные для CRM и обучения команды`

На следующем этапе проект можно развивать в сторону `Consultation Copilot`, то есть live-помощника диагноста во время консультации.
