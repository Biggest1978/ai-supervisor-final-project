# Текст для сдачи ДЗ

Здравствуйте.

В качестве итогового проекта я сдаю `AI Supervisor` — рабочую агентную систему для анализа диагностическо-продажных консультаций.

Проект решает реальную прикладную задачу: помогает разбирать консультации, выявлять ошибки экспертов, формировать рекомендации по следующему шагу, собирать отчетность и возвращать структурированные выводы в будущий CRM-контур.

Это не учебный toy-project, а MVP реального рабочего контура. В более широком смысле `AI Supervisor` является частью проекта `CRM Dream` / цифровой экосистемы отдела диагностик.

Что реализовано в MVP:

- проектная агентная логика;
- маршрутизация задач по ролям;
- структура выходных артефактов;
- логика review и отчетных документов;
- связь с более широким Sales OS / CRM-контуром.

Коротко о результате:

`AI Supervisor` переводит контроль качества консультаций из ручного и точечного режима в системный: помогает видеть повторяющиеся ошибки, усиливать экспертов и накапливать полезную аналитическую базу для дальнейшей автоматизации.

В репозитории я приложил:

- описание проекта;
- архитектуру агентной системы;
- материалы для краткой защиты;
- чек-лист формальной сдачи.
